# Alphonso Ecosystem — Codex Handoff Package

This package organizes the current Alphonso ecosystem direction into clean files for Codex, Gemini, or another implementation agent.

## Contains
- Project overview
- Codex implementation prompts
- Gemini UI prompts
- Component Factory prompt
- Agent identity docs
- UI/UX system notes
- Lore Bible update notes
- Mascot/image assets generated or referenced in this session
- Build verification checklist
- Roadmap and next implementation order

## Current agent model
```text
Jose      = Orchestrator / Governor / Workload Router
Alphonso  = Operator / Runtime / Verification / Execution
Miya      = Creator / Media / Marketing / Storytelling
```

## Recommended workflow
```text
Gemini or UI AI = designs and generates isolated components
Codex = integrates, verifies, builds, tests
ChatGPT = architecture, debugging, orchestration, prompts, strategy
```

- Tauri v2 only
- React frontend
