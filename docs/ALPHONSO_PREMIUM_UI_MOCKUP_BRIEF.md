# ALPHONSO Premium UI Mockup Brief

Goal: redesign ALPHONSO from a generic dark dashboard into a premium AI mission-control product.

## Product Feel

- Private local AI command center, not SaaS template.
- Premium, cinematic, calm, operational.
- Feels like a personal AI headquarters: ALPHONSO + Jose + Miya + Coach + Operator.
- Strong identity, fewer generic cards, more hierarchy and atmosphere.

## Visual Inputs

Use the existing brand/media direction from:

- `logo-banner-thumbnail-media/ALPHONSO_ICON.png`
- `logo-banner-thumbnail-media/ALPHONSO_LOGO.png`
- `logo-banner-thumbnail-media/ALPHONSO_BANNER.png`
- `logo-banner-thumbnail-media/ALPHONSO_THUMBNAIL.png`

Important: fix/avoid any AI-text artifacts from raw generated banners.

## Typography

Move away from generic dashboard font feel. Explore premium technical fonts:

- Sora
- Space Grotesk
- Manrope
- Geist
- Plus Jakarta Sans
- Inter Tight

Recommended direction: Sora or Space Grotesk for headings, Manrope/Geist for UI text.

## Screens To Mock

1. Mission Control Home
   - hero command center
   - current mission
   - next 3 actions
   - live pulse/feed
   - Coach state
   - Jose approval state
   - Railway/deploy state

2. Chat Hub
   - premium compact chat
   - agent output rail
   - less generic bubbles
   - clear active model/local/cloud lane

3. Jose Orchestrator
   - command intake as cockpit, not form
   - critical panels only by default
   - collapsible operations/history

4. Coach Overlay
   - protective, direct, noticeable
   - hard interrupt should feel serious but not scary

## Components To Design

- Premium app shell/sidebar
- Mission hero/header using ALPHONSO identity
- Next Action cards
- Agent status cards
- Unified activity feed
- Hard interrupt overlay
- Connector status chips
- Focus/Full density toggle

## Avoid

- Generic SaaS dashboard cards
- Too many equally weighted boxes
- Random gradients without purpose
- Overcrowded telemetry
- Fake 3D clutter
- AI-generated unreadable text

## Mockup Prompt

Design a premium dark-mode web app called ALPHONSO: a private local AI mission-control command center for a personal AI ecosystem. It should feel cinematic, sharp, calm, and operational — not a generic SaaS dashboard. Use a deep black/graphite base, cyan/amber/fuchsia accents, subtle glass panels, crisp hierarchy, and premium typography such as Sora/Space Grotesk/Manrope. Include a Mission Control Home screen with a hero command-center area, current mission, next 3 recommended actions, live activity pulse, Coach guardian state, Jose approval state, and connector/deploy status. Integrate ALPHONSO brand imagery/logo direction tastefully, but avoid clutter and fake unreadable AI text. Prioritize whitespace, strong section hierarchy, and distinctive product identity.
