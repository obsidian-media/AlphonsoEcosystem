### Running the app in dev mode
```powershell
# Kill anything on port 5173 first if you get a port conflict
$proc = Get-NetTCPConnection -LocalPort 5173 -ErrorAction SilentlyContinue | Select-Object -First 1 -ExpandProperty OwningProcess
if ($proc) { Stop-Process -Id $proc -Force }

npm run tauri dev
```

### Building a new installer
```powershell
npm run build           # verify frontend first
cd src-tauri; cargo check   # verify Rust
cd ..
npx tauri build         # produces NSIS + MSI in src-tauri/target/release/bundle/
```

### Cutting a signed release
```powershell
$env:TAURI_SIGNING_PRIVATE_KEY = "path\to\alphonso.key"
$env:ALPHONSO_UPDATE_BASE_URL = "https://your-release-host/alphonso/windows"
npm run release:updater
```
